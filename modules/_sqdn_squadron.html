CommHandler

CommHandler: CommReceiver | CommResponder | CommProcessor | CommProtocol
  • Defined in sqdn/squadron/src/communication/comm-handler.ts:18

Inbound command handler.

One of:

  • signal receiver,
  • request responder,
  • command processor, or
  • communication protocol.

A Communicator processes incoming commands by handlers available in unit context as CommProtocol.
