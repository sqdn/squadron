Interface Target

A message to process before being logged by logger.

Has the same structure as DueSqdnLog but some properties may be initially omitted.

Optional index

index?: number

Inherited from DueLog.Target.index

  • Defined in proc7ts/logger/index.d.ts:116

An index of the first element of the log line to process.

Defaults to 0. When set to negative value, reset to 0. When set to the value greater than the length of the log line, reset to the log line length.

After processing will be set to the length of the final log line.

zDetails?: Record<string | symbol, unknown>
  • Defined in sqdn/squadron/src/logging/due-sqdn-log.ts:45

Log message details to process and log.

Can be modified or replaced to change the final message details.
