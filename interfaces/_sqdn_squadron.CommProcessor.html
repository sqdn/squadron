receive

  • receive(name: string, signal: CommPacket): boolean
    • Defined in sqdn/squadron/src/communication/comm-processor.ts:25

    Handles received signal.

    May skip signal processing. In this case a false value should be returned. The next receiver in processing chain will receive the signal then.

    Parameters

    • name: string

      Received signal name.

    • signal: CommPacket

      Received signal data packet.

    Returns boolean

    Either true if the signal processed, or false if unknown signal received.
